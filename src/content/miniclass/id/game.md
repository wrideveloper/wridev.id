## Pengenalan Singkat

Ubah hobimu dari sekadar **“pemain”** menjadi **“pencipta”**. Di **Miniclass Game Development**, kamu akan menyelami seluk-beluk pembuatan game menggunakan tools standar industri: **Unity** sebagai game engine, serta **Blender** dan **Aseprite** untuk pembuatan aset visual. Pengalaman belajar yang kamu dapatkan dirancang agar menyerupai workflow developer game profesional.

Materi pelatihan berfokus pada **praktik langsung**, dan uniknya kamu bisa memilih jalur belajarmu sendiri melalui sistem **Sub-Class**. Apakah kamu tipe logis yang ingin mengendalikan mekanik game lewat koding **C#**? Atau kamu tipe kreatif yang ingin menciptakan karakter **2D atau 3D** yang ikonik? Materi akan disesuaikan dengan peran yang kamu minati, mulai dari **logic scripting** hingga **asset creation**.

Tidak perlu khawatir jika kamu belum pernah membuat game sebelumnya. Miniclass ini dirancang khusus untuk **pemula (Beginner)**, dengan mentor yang siap membimbingmu memahami proses pembuatan game dari nol. Kamu akan belajar mengikuti workflow industri, namun tetap dengan suasana santai tanpa tekanan berlebih.

Sebagai penutup, kamu akan merasakan simulasi bekerja di **studio game sungguhan**. Pada pertemuan akhir (**Game Jam**), kamu akan berkolaborasi dalam tim yang terdiri dari programmer dan 2D/3D artist untuk menggabungkan kode dan aset visual menjadi sebuah **prototype game yang dapat dimainkan**. Di sini, kamu tidak hanya mengasah skill individu, tetapi juga belajar bekerja sama dalam sebuah tim.

Jadi, tunggu apa lagi? Bergabunglah dengan **Miniclass Game Development**, jelajahi luasnya dunia game, pilih peranmu, dan ciptakan dunia impianmu sendiri!

## Timeline Pembelajaran

### Pertemuan 1: Selamat Datang di Game Development
**Fokus:** Pengenalan Umum & Ekosistem *(Semua Role Gabung)*  
**Topik Utama:** Alur kerja Game Development, tren industri game di Indonesia, dan pengenalan peran dalam tim  
**Output:** Peserta memahami peran dan tanggung jawab tiap anggota tim  

### Pertemuan 2: Fondasi & Aspek Game
**Fokus:** Membedah elemen pembentuk game dan persiapan aset dasar  

**Sub-Class: Programmer (Unity Essentials)**  
- Mempelajari aspek gameplay dan logika dasar  

**Sub-Class: 2D Artist (Asset Rules)**  
- Pengenalan dasar aset game 2D (Pixel Art / Vektor) menggunakan Aseprite  

**Sub-Class: 3D Artist (Basic 3D Modeling)**  
- Pengenalan UI/UX Blender dan dasar-dasar 3D Modeling  

### Pertemuan 3: Implementasi & Aturan Aset
**Fokus:** Pengenalan teknis tools sesuai role  

**Sub-Class: Programmer (Unity Essentials)**  
- Pengenalan fitur utama Unity Game Engine  
- Navigasi editor, GameObjects, dan Components  

**Sub-Class: 2D Artist (Asset Rules)**  
- Aturan teknis aset 2D (Symmetry, Color Palette, Power of Two)  
- Membuat sprite sederhana yang siap digunakan di engine  

**Sub-Class: 3D Artist (Basic 3D Modeling)**  
- Praktik membuat model 3D dasar (Blocking)  
- Memahami koordinat dan transformasi objek  

### Pertemuan 4: Pendalaman Materi (Intermediate)
**Fokus:** Membangun fungsi dan detail karakter  

**Sub-Class: Programmer (Scripting 101)**  
- Membuat script pertama menggunakan C#  
- Logika pergerakan karakter dan interaksi sederhana  

**Sub-Class: 2D Artist (Moodboard & Concept)**  
- Membuat moodboard untuk menentukan gaya visual game  
- Mendesain character concept art yang lebih matang  

**Sub-Class: 3D Artist (Intermediate Modeling)**  
- Teknik perataan mesh (Retopology dasar)  
- Pemberian material dan persiapan aset untuk export ke Unity  

### Pertemuan 5: Game Jam (Final Project)
**Fokus:** Kolaborasi & Output *(Semua Role Gabung)*  
**Kegiatan Utama:**  
- Pembentukan tim kecil yang terdiri dari Programmer, 2D Artist, dan 3D Artist  

**Tujuan:**  
- Menggabungkan seluruh aset dan kode menjadi satu prototype game sederhana  

**Output:**  
- Pengalaman kerja tim (soft skill)  
- Prototype game dan karya akhir untuk portofolio
