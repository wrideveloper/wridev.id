## Pengenalan Singkat

Program pelatihan bagi mahasiswa TI, **Miniclass Backend Development** mempelajari dasar-dasar pengembangan logika sistem menggunakan platform **PHP dan MySQL** sesuai dengan standar industri. Di kelas ini, kamu akan belajar skill fundamental yang pasti terpakai di dunia kerja.

Kamu mungkin berpikir, *“ah backend mah ngebosenin, cuma baris kode doang.”* Tapi di sini, kita mengutamakan **pendekatan praktis**, di mana kamu akan langsung mencoba membangun logika sistem. Mulai dari merancang database yang rapi, menghubungkan website dengan server, hingga membuat fitur interaktif seperti manipulasi data (**Create, Read, Update, Delete**) dan logika relasi data yang lebih kompleks.

Jangan khawatir kalau kamu belum pernah ngoding sebelumnya. **Miniclass Backend** dirancang untuk pemula (*Beginner*). Entah kamu baru pertama kali dengar istilah *“Query”* atau *“HTTP Method”*, mentor kami siap membimbingmu langkah demi langkah. Kamu akan belajar logika pemrograman tanpa tekanan, tapi tetap dengan hasil yang memuaskan.

Yang paling seru, kamu akan memahami bagaimana data bekerja di dunia nyata. Lewat berbagai studi kasus, kamu tidak hanya melatih logika berpikir, tapi juga mendapatkan gambaran nyata tentang pekerjaan sebagai seorang **Backend Developer**.

Jadi, tunggu apa lagi? Bergabunglah dengan **Miniclass Backend** dan jadilah programmer andal di balik layar aplikasi-aplikasi hebat!

## Timeline Pembelajaran

### Pertemuan 1
**Topik Utama:** Pengenalan Database, Setup Database, DDL  
**Output:**
- Peserta mampu melakukan setup database menggunakan Laragon  
- Peserta mampu membuat query DDL melalui phpMyAdmin  

### Pertemuan 2
**Topik Utama:** Pengenalan PHP, DML, Database Connection  
**Output:**
- Peserta mampu membuat query DML melalui phpMyAdmin  
- Peserta mampu membuat koneksi database ke proyek PHP  

### Pertemuan 3
**Topik Utama:** Pengenalan HTTP Method, CREATE dan READ Data  
**Output:**
- Peserta memahami konsep HTTP Method  
- Peserta mampu mengimplementasikan method POST untuk CREATE data pada PHP  
- Peserta mampu mengimplementasikan method GET untuk READ data pada PHP  

### Pertemuan 4
**Topik Utama:** UPDATE dan DELETE Data  
**Output:**
- Peserta mampu mempraktikkan UPDATE data pada PHP  
- Peserta mampu mempraktikkan DELETE data pada PHP  

### Pertemuan 5
**Topik Utama:** Constraint, Relationship, Query JOIN  
**Output:**
- Peserta memahami konsep constraint  
- Peserta memahami relationship antar tabel  
- Peserta mampu membuat query JOIN pada phpMyAdmin  

### Pertemuan 6
**Topik Utama:** Studi Kasus JOIN dengan PHP  
**Output:**
- Peserta mampu melakukan JOIN antara tabel `barang` dan `detail_penjualan` lalu menampilkannya di website  
- Peserta mampu melakukan JOIN antara tabel `barang` dan tabel `kategori`
