## Pengenalan Singkat

Program pelatihan bagi mahasiswa TI, **Miniclass Frontend Development** mempelajari dasar-dasar pengembangan antarmuka pengguna (*user interface*) pada platform **Web dan Mobile** sesuai dengan standar industri. Dalam pelatihan ini, kamu akan menggunakan tools yang umum dipakai oleh developer profesional seperti **VS Code, Web Browser, dan Flutter SDK**.

Selama pelatihan, kamu akan langsung mempraktikkan teori yang dipelajari sehingga siap untuk terjun ke dunia profesional. Mulai dari menyusun struktur website, mendesain tampilan aplikasi mobile yang responsif, hingga membuat halaman interaktif dengan logika pemrograman. Semua materi disusun secara bertahap dan saling terhubung.

Tidak perlu khawatir jika kamu belum pernah ngoding sebelumnya. **Miniclass Frontend Development** dirancang khusus untuk pemula (*Beginner*). Tim Frontend kami akan membimbingmu langkah demi langkah. Kamu akan belajar dengan suasana santai, tanpa tekanan, namun tetap mendapatkan skill bernilai tinggi yang sangat dibutuhkan di dunia kerja.

Di kelas ini, kamu juga akan belajar layaknya developer profesional. Tidak hanya menulis kode, tetapi juga mengelola proyek menggunakan **Git** dan tools pendukung lainnya. Dengan begitu, kamu tidak hanya memiliki kemampuan teknis, tetapi juga memahami cara kerja yang rapi, terstruktur, dan profesional.

Jadi, tunggu apa lagi? Bergabunglah dengan **Miniclass Frontend Development**, siapkan VS Code-mu, dan mari ubah barisan kode menjadi tampilan aplikasi yang keren dan hidup!

## Timeline Pembelajaran

### Pertemuan 1: Fondasi Dasar & Instalasi
**Fokus:** Membangun fondasi paling dasar di kedua platform  
- **Mobile:** Dasar Bahasa Dart (Variabel, Tipe Data) dan Instalasi Flutter  
- **Web:** Struktur HTML (DOCTYPE, head, body) dan Tag Dasar

**Fokus:** Web pada struktur lanjutan, mobile pada logika fungsi  
- **Mobile:** Perulangan (for, while) dan penggunaan fungsi dasar di Dart  
- **Web:** Form, Tabel, Elemen Semantik, dan CSS Dasar (Selector, Box Model)  

### Pertemuan 3: Tata Letak & Struktur Data üìê
**Fokus:** Web mulai styling, mobile mulai menstrukturkan data  
- **Mobile:** Pengenalan Class di Dart dan Collections (List, Set, Map)  
- **Web:** Tata letak modern menggunakan Flexbox, Grid, dan Media Query  

### Pertemuan 4: UI Framework & JavaScript Basics üé®
**Fokus:** Web menata layout, mobile masuk ke UI framework  
- **Mobile:** Widget dasar (Scaffold, AppBar, Container) dan styling dasar  
- **Web:** Pengenalan JavaScript (var, let, const, data types, functions)

**Fokus:** Membuat UI menjadi dinamis dan mengenal navigasi
