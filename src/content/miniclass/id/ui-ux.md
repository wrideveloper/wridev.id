## Pengenalan Singkat

**Miniclass UI / UX Design** adalah program pelatihan yang mengupas tuntas dunia desain produk digital dengan menggabungkan aspek **visual (UI)** dan **pengalaman pengguna (UX)**. Dalam kelas ini, kamu akan menggunakan **Figma**, salah satu tools standar industri saat ini, sehingga skill yang kamu pelajari akan sangat relevan untuk dunia kerja profesional.

Materi pelatihan dirancang agar mudah dipelajari, mulai dari nol hingga mahir. Kamu akan mempelajari berbagai fitur penting di Figma seperti **Auto Layout** dan **Components** yang membuat proses desain menjadi lebih efisien. Materi mencakup penyusunan **Style Guide**, pembuatan **Design System sederhana**, hingga pembuatan **Prototype interaktif**.

Tidak perlu minder jika kamu merasa tidak jago menggambar atau tidak memiliki latar belakang seni. **Miniclass UI / UX** ini dirancang khusus untuk pemula (*Beginner*). Mentor akan membimbingmu memahami perbedaan antara **UI (tampilan)** dan **UX (pengalaman)**, serta mengenalkan metode **Design Thinking** agar desain yang kamu buat benar-benar menjawab kebutuhan pengguna.

Desain yang baik bukan hanya soal tampilan yang indah, tetapi juga tentang kenyamanan dan kemudahan penggunaan. Di kelas ini, kamu akan belajar membuat desain yang rapi, konsisten, dan sesuai dengan standar profesional. Skill UI / UX merupakan salah satu kemampuan penting yang sangat dicari di industri teknologi saat ini.

Jadi, tunggu apa lagi? Yuk bergabung di **Miniclass UI / UX Design**! Siapkan ide-ide kreatifmu dan mari kita rancang pengalaman digital masa depan yang menarik, solutif, dan berdampak.

## Timeline Pembelajaran

### Pertemuan 1: Pengenalan UI/UX Design & Tools
**Fokus:** Fondasi UI/UX dan Tools Dasar  
**Topik Utama:**  
- Pengenalan UI vs UX  
- Peran UI/UX Designer  
- Instalasi dan setup Figma  
- Workspace dan fitur dasar Figma  

**Praktikum:**  
- Install Figma  
- Eksplor workspace  
- Mengenal tools dasar (frame, shape, text, constraint)  

**Output:**  
- Peserta memahami dasar UI/UX dan siap menggunakan Figma  

### Pertemuan 2: Layouting Dasar UI
**Fokus:** Membiasakan penggunaan Figma  
**Topik Utama:** Layouting dasar, alignment, spacing, dan visual hierarchy  
**Praktikum:** Mendesain UI card tanpa auto layout dengan fokus kerapian  
**Output:** UI card dengan alignment dan spacing yang baik  

### Pertemuan 3: Auto Layout & Responsive Design
**Fokus:** Best practice dalam membuat desain  
**Topik Utama:** Auto Layout dan konsep responsive design  
**Praktikum:**  
- Menerapkan auto layout  
- Mengubah desain menjadi responsif  

**Output:** UI yang fleksibel dan scalable  
**Goals:** Memahami dan menerapkan auto layout dengan benar  

### Pertemuan 4: UI Theory, Gestalt Principle & UI Fundamental
**Fokus:** Prinsip Dasar UI  
**Topik Utama:** UI Theory, Gestalt Principle, dan UI Fundamental  
**Praktikum:**  
- ATM (Amati, Tiru, Modifikasi)  
- Menerapkan Gestalt Principle dan UI Fundamental  

**Output:** UI yang lebih terstruktur dan mudah dipahami  

### Pertemuan 5: Grid, Color, Typography & Style Guide
**Fokus:** Konsistensi Visual  
**Topik Utama:** Grid system, teori warna, tipografi, style guide, dan design system  
**Praktikum:** Membuat color styles dan text styles  
**Output:** Style guide dasar untuk desain  

### Pertemuan 6: Component & Variant Dasar
**Fokus:** Reusable Design  
**Topik Utama:** Konsep component dan variant  
**Praktikum:** Membuat komponen dengan state hover, focus, disabled, dan lainnya  
**Output:** Komponen reusable siap digunakan  

### Pertemuan 7: Prototyping Dasar
**Fokus:** Interaksi dan Alur  
**Topik Utama:** Prototyping di Figma  
**Praktikum:** Menghubungkan antar screen menggunakan template yang disediakan  
**Output:** Prototype interaktif  

### Pertemuan 8: Design Process Dasar
**Fokus:** Alur Design Process  
**Topik Utama:** Design process (brief → competitive analysis / benchmarking → user flow)  
**Praktikum:**  
- Memahami brief dari mentor  
- Melakukan competitive analysis  
- Membuat user flow  

**Output:** User flow dari studi kasus  

### Pertemuan 9: Metode Design Process
**Fokus:** Pendekatan dan Framework Desain  
**Topik Utama:**  
- Design Thinking  
- Double Diamond  
- Lean UX  

**Praktikum:** Diskusi dan pemetaan metode design process sesuai studi kasus  
**Output:** Peserta memahami konteks dan perbedaan tiap metode  

### Pertemuan 10: Empathize, Define, Ideate & Information Architecture
**Fokus:** UX Research dan Problem Definition  
**Topik Utama:** Empathize, Define, Ideate, dan Information Architecture  
**Praktikum:**  
- Menyusun pertanyaan survei  
- Membuat *How Might We* question  
- Menghasilkan ide solusi  

**Output:** Problem statement dan ide solusi berbasis user  

### Pertemuan 12: Testing & Validasi Desain
**Fokus:** UX Testing  
**Topik Utama:** Testing dalam UX Design  
**Praktikum:** Membuat skenario usability testing menggunakan Maze  
**Output:** Skenario testing dan insight awal dari user testing
